iter-together
================

A package to iterate over multiple iterables together


Installation
------------

To download this code and install it in development mode, run the following:

.. code-block:: bash

    $ git clone https://github.com/commons-research/iter-together.git
    $ cd iter-together
    $ python install -e .


Testing
-------

.. code-block:: bash

    $ pip install tox
    $ tox
